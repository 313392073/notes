#####垂直水平居中有多重方式：
	
####1、fixed定位，设定top,left,bottom,right,margin。
	元素显示在屏幕中央
	IE不支持
	```
    .one{
		width: 45px;
		height: 45px;
		position: fixed;
		top: 0;
		right: 0;
		bottom: 0;
		left: 0;
		margin: auto;
	}
	```
	
####2、绝对定位，在设置top,left,margin-top,margin-left。
	元素显示在父级relative定位的元素中央。
	需要固定宽高
	```
	.two{
		position: absolute;
		top:50%;
		left:50%;
		width: 40px;
		height: 40px;
		background: #000;
		margin-top: -20px;
		margin-left:-20px;
	}
	```
####3、利用position:absolute属性，设置top/bottom/right/left
	元素显示在父级relative定位的元素中央。
	```
	.three{
		position: absolute;
		width: 140px;
		height: 140px;
		top: 0;
		right: 0;
		bottom: 0;
		left: 0;
		margin: auto;
	}
	```
####4、利用display:table-cell属性使内容垂直居中(用于父元素)
	```
	.four{
		display:table-cell;
		vertical-align:middle;
	}
	```
####5、使行内元素居中，line-height(用于父元素)
	```
	.five{
		width:100px;
		height:100px;
		line-height:100px;
	}
	```
####6、使用css3的新属性transform:translate(x,y)属性
	```
	.six{
		position: absolute;
		width: 40px;
		height: 40px;
		background: #000;
		top: 50%;
		left: 50%;
		transform: translate(-50%,-50%);
		-webkit-transform:translate(-50%,-50%);
		-moz-transform:translate(-50%,-50%);
		-ms-transform:translate(-50%,-50%);
	}
	```
